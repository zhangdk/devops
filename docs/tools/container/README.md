# 容器化和编排工具

## 1. 容器技术基础

### 1.1 Docker
- **核心概念**
  - 镜像（Image）
  - 容器（Container）
  - 仓库（Registry）
  - Dockerfile

- **主要特性**
  - 轻量级虚拟化
  - 快速部署
  - 版本控制
  - 可移植性

### 1.2 容器运行时
- **containerd**
  - OCI标准实现
  - 高性能
  - 稳定性好
  - 广泛应用

- **CRI-O**
  - Kubernetes原生
  - 轻量级
  - 安全性高
  - 标准兼容

  - 服务发现

### 2.2 Docker Swarm
- **特点**

- **核心功能**
