## 2. 主流监控工具

### 2.1 Prometheus
- **核心特性**
  - 时序数据库
  - 强大的查询语言
  - 灵活的数据模型
  - 高效的存储

- **主要组件**
  - Prometheus Server
  - Alertmanager
  - Exporter
  - Pushgateway

### 2.2 Grafana
- **核心特性**
  - 可视化展示

- **主要功能**
  - 仪表盘

  - 告警规则

  - 实时监控
